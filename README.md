# 🌸 Bloom Planner — Flutter App

> A rolling perpetual calendar app that feels like a Happy Planner, Bullet Journal, and Life Dashboard rolled into one impossibly cute package.

---

## ✨ Features

- **Rolling Calendar** — dates auto-advance, no yearly reset
- **Habit Tracker** — streaks, weekly grid, heatmap
- **Goal Planner** — annual → quarterly → monthly → daily cascade
- **Vision Board** — photo + emoji tiles
- **Journal** — gratitude, intention, reflection sections with stickers
- **Water Tracker** — tap-to-log cups
- **Mood Logger** — 5-point scale with daily history
- **Family Hub** — shared calendar, member profiles
- **Budget Planner** — income / expense categories
- **ADHD-Friendly Mode** — 3-task focus, time blindness alerts, dopamine rewards
- **Seasonal Themes** — spring 🌸 / summer ☀️ / autumn 🍂 / winter ❄️
- **Dark Mode** — full Material 3 dark theme
- **Offline-First** — Isar local DB, syncs to Firebase when online
- **Achievement Badges** — unlock milestones, celebrate streaks

## 🚀 Getting Started

### Prerequisites

```bash
flutter --version   # >= 3.19.0
dart --version      # >= 3.3.0
```

### 1. Clone & install dependencies

```bash
git clone <your-repo>
cd bloom_planner
flutter pub get
```

### 2. Set up Firebase

```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Log in to Firebase
firebase login

# Configure — this overwrites lib/firebase_options.dart
flutterfire configure
```

### 3. Generate Isar models

```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

> This generates `lib/core/db/models.g.dart` from the `@collection` annotations.

### 4. Add fonts

Download and place in `assets/fonts/`:
- **Playfair Display** — [Google Fonts](https://fonts.google.com/specimen/Playfair+Display)
- **Caveat** — [Google Fonts](https://fonts.google.com/specimen/Caveat)
- **DM Sans** — [Google Fonts](https://fonts.google.com/specimen/DM+Sans)

Files needed:
```
assets/fonts/
├── PlayfairDisplay-Regular.ttf
├── PlayfairDisplay-Bold.ttf
├── PlayfairDisplay-Italic.ttf
├── Caveat-Regular.ttf
├── Caveat-Bold.ttf
├── DMSans-Regular.ttf
├── DMSans-Medium.ttf
└── DMSans-Bold.ttf
```

### 5. Run

```bash
# iOS simulator
flutter run -d iPhone

# Android emulator
flutter run -d emulator-5554

# Web (for dev preview)
flutter run -d chrome
```

## 🔑 Environment Variables

For CI/CD, inject Firebase keys via `--dart-define`:

```bash
flutter build ios \
  --dart-define=FIREBASE_API_KEY=xxx \
  --dart-define=FIREBASE_APP_ID=xxx
```

Never commit `firebase_options.dart` with real keys to a public repo.
